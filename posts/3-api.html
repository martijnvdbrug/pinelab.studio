API development

Ik ben begonnen als Java developer bij Incentro, waar ik ook mijn Google Certified Professional Cloud Architect haalde. 2 jaar gelden ben ik als NodeJS developer verder gegaan bij MisterGreen, waar ik ontdekt heb dat de toekomst van API development bij GraphQL ligt.

GraphQL, NodeJS, Apollo Server en NestJS op Google Cloud

De favoriete stack is vandaag de dag een GraphQL API ontwikkeld in NodeJS. Hiervoor gebruik ik Apollo Server als GraphQL framework met NestJS als dependency injection framework. Dit deploy ik op vervolgens Google Cloud Run.

Minimum Viable Product

Deze flexibele stack leent zich perfect om een Minimum Viable Product (MVP) op te zetten, zonder al te veel overhead. Dat gezegd hebbende, kan deze stack ook prima productiewaardige software draaien. Neem bijvoorbeeld het MisterGreen platform, waar door gebruikers dagelijks ~1000 Tesla's geconfigureerd worden.

Case: MisterGreen

In dienst van MisterGreen heb ik, samen met 2 andere developers, het MisterGreen Electric lease platform opgezet. Dit platform maakt het mogelijk voor klanten om online een Tesla te configureren en te bestellen. Dit platform heeft een GraphQL API, ontwikkeld in NodeJS, met de frameworks NestJS en Apollo Server. Meer technische info is hier te vinden.

Case: Pinelab Webshop

Om aan de vraag naar kleine webshops te voldoen, heb ik een SaaS eCommerce platform opgezet. Uiteraard word ook hier weer gebruik gemaakt van mijn favoriete stack: GraphQL en NodeJS. De backend bestaat deels uit het Open Source headless eCommerce platform Vendure, met een aantal custom plugins.
